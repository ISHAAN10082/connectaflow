# Connectaflow

Connectaflow is a modular CRM and Lead Management System designed to streamline lead processing, enrichment, and management. It features a FastAPI backend for data processing and a Next.js frontend for a modern, responsive user interface.

## 🚀 Quick Start (Non-Developers)

If you just want to run the application, please refer to the [How to Run Guide](docs/how_to_run.md).

## 🛠️ Developer Guide

For developers looking to contribute or understand the codebase, please refer to the [Developer Guide](docs/developer_guide.md).

## 📁 Project Structure

- **backend/**: FastAPI application handling logic, database, and API.
- **frontend/**: Next.js application for the user interface.
- **assets/**: Static assets and data files.
- **scripts/**: Utility scripts.
- **docs/**: Documentation.
